# cursor-practice

Cursor（エディタ＋AI）の扱い方を、**基礎から発展的な使い方まで**身につけたい、という学習用のリポジトリです。実験用の小さなコードや、動かして試す用の置き場として使っています。サンプル UI は、紙本・電子籍の**読書記録（一覧・状態・感想・日付。`localStorage` 永続化）**を扱います。以前入っていたメモアプリ相当の例は、**読書用アプリの実装に置き換える方針**です（併用しません）。データは **`reading-journal:books` キー**にのみ JSON で保存し、**旧 `memo-app:items`（メモ用キー）は使いません**（ドメイン分離＆重複表示を避けるため。キー名は [context/main/task.md](context/main/task.md) および実装内の定数に合わせてください）。

## 目安

- このフォルダ内のプロジェクトを **書き換えたり、AI に聞きながら拡張したり** して、チャット・エージェント・コマンド・ルールなど、Cursor らしいワークフローを試す
- 慣れてきたら、**自分の学びたい技術**（例: フロント、CLI、型の書き方）に寄せた練習課題を足していく、という進め方がおすすめです

厳格な完成品を作る目的ではなく、**触って理解を深める場所**、という立ち位置のリポジトリです。
